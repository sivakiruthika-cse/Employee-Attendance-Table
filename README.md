# Employee Attendance Table

## Description
This project is a simple and visually styled Employee Attendance Table created using HTML and CSS. It displays employee details along with their attendance status in a structured tabular format. The design includes hover effects, color-coded status indicators, and pseudo-elements for better visual representation.

## Features
- Structured attendance table using HTML
- External CSS styling
- Gradient background design
- Hover effect on table rows
- Color-coded attendance status (Present / Absent)
- Pseudo-elements with check (✔) and cross (✘) symbols
- Clean and centered layout

## Technologies Used
- HTML5
- CSS3

## Sections Included
- Employee ID
- Employee Name
- Department
- Attendance Status

## Purpose
This project demonstrates understanding of:
- HTML table structure (thead, tbody, caption)
- CSS styling and layout design
- Pseudo-elements (::before)
- Hover effects and visual enhancements

## How to Run
1. Download or clone the repository.
2. Ensure both `index.html` and `styles.css` files are in the same folder.
3. Open `index.html` in any web browser.
4. The Monthly Attendance Table will be displayed.
